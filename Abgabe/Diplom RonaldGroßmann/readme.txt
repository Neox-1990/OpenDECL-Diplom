-data
	-Konfigurationen: Alle getestet Konfigurationen auf den verschiedenen LHRD-Installationen
	-XML-Spezifikation: Die Spezifikation von OpenDECL als XSD-Datei, die OpenDECL-Beschreibungen der getesteten Installationen, die Beispiel-XSLT-Dateien zum Anwenden auf die OpenDECL-Beschreibungen
	
-project
	-bin: ausführbarer Editor (OpenDECL Editor.jar), wobei "default-min.xml" benötigt wird
	-doc: kurze Anleitung, aus Kapitel 7 der Ausarbeitung
	-src: Quelldateien, in editor die entsprechenden Klassen für den Editor, in xmleditorkit eine Bibliothek für das Syntaxhighlighting

-thesis: Ausarbeitung als PDF, sowie die LaTex Quelldateien und Bilder
